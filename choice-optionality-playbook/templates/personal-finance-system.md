# 个人财务系统表

> 这不是一张“净资产截图”，而是一套持续回答三个问题的表：我现在能撑多久？最脆弱的一环是什么？我怎样用收入、资产和生活设计，把跑道拉长？

配套表格包含八个工作表，可直接在 Google Sheets 中建立并长期维护。

## 工作表结构

| 工作表 | 核心用途 | 关键输出 |
|---|---|---|
| 总览 | 汇总当前状态与行动优先级 | 当前 / 压缩 / 最低成本、可用资产、跑道 |
| 预算 | 将支出拆成生存层、维持层、可选层 | 三套月生活成本 |
| 资产 | 将账面资产转换为压力折扣后的可用资产 | 30 天可动用资产、折扣后资产 |
| 债务与承诺 | 记录贷款、长期租约、家庭责任和保险 | 月刚性支出、到期 / 复盘日 |
| 收入与现金流 | 区分工资、稳定收入、投资现金流 | 最低成本覆盖率 |
| 目标与资产路径 | 把当前系统连接到未来目标 | 目标缺口、阶段性行动 |
| 决策日志 | 记录重大投资、职业、迁移和消费决策 | 假设、失效条件、复盘结果 |
| 说明 | 固化维护节奏和边界 | 每月 / 每季复盘清单 |

## 关键公式

```text
最低生存跑道（月）
= 30 天内可动用的低风险资产
÷ 最低月生存成本

压力折扣后可用资产
= 当前市值 × 压力折扣

最低成本覆盖率
= 稳定月收入 ÷ 最低月生存成本

目标缺口
= 目标金额 - 当前可用资产
```

## 输入规则

- **蓝色数字**：由使用者填写或更新，例如支出、资产市值、债务余额、利率和收入。
- **黑色数字**：公式自动计算，不手动覆盖。
- 不把自住房、高波动仓位、期权、锁定资产或自己实际不会卖出的长期仓位，全额算进生存缓冲。
- 未实现浮盈不是可持续现金流；不要用短期行情扩大固定生活成本。

## 使用顺序

1. 在 `预算` 中填写当前、压缩、最低三套支出；
2. 在 `资产` 中填写市值、可动用性和压力折扣；
3. 在 `债务与承诺` 中填入月最低支出、利率、到期日和退出成本；
4. 回到 `总览`，观察最低生存跑道与最脆弱环节；
5. 再使用[个人财务选择权计算器](https://hawtim.github.io/personal-finance-calculator/)测试 5 年 / 10 年未来资产路径；
6. 每月更新一次数据；每季度做一次压力测试与目标复盘。

## 建议的压力测试

至少测试以下四种情况：

1. 可投资资产下跌 30%；
2. 未来 12 个月没有工资收入；
3. 出现相当于 6 个月最低生活成本的医疗或家庭支出；
4. 居住成本短期无法下降。

目的不是预测灾难，而是确认：在坏情景出现时，你是否仍保有决策能力。
