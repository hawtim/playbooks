# 职业退出权测算表

这不是“达到一个数字就辞职”的计算器，而是用来衡量你在不同情景下拥有多少时间。

## 输入

### 支出

| 项目 | 当前生活 | 压缩生活 | 最低生存 |
|---|---:|---:|---:|
| 居住 |  |  |  |
| 饮食 |  |  |  |
| 医疗与保险 |  |  |  |
| 交通与通信 |  |  |  |
| 债务最低还款 |  |  |  |
| 家庭/宠物责任 |  |  |  |
| 工作与社交维持 |  |  |  |
| 月合计 |  |  |  |

### 可用资产

| 类别 | 金额 | 压力折扣 | 计入缓冲 |
|---|---:|---:|---:|
| 现金/活期 |  | 100% |  |
| 货币基金/短存 |  | 95%–100% |  |
| 低波动资产 |  | 80%–95% |  |
| 长期指数仓 |  | 50%–80% |  |
| 高波动仓/期权/加密 |  | 0%–40% |  |
| 房产/非标资产 |  | 0%–30% |  |

## 输出

```text
生存月数 = 可计入缓冲金额 ÷ 最低月生存成本
压缩跑道 = 可计入缓冲金额 ÷（压缩月成本 - 可持续非工资现金流）
```

## 压力测试

至少模拟：资产下跌 30%、12 个月无工资、6 个月最低生活成本的突发支出、居住成本无法立刻下降，以及转型所需的学习或搬家成本。
